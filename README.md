# ChoanoCounter

For all your choano counting needs!

## Installation
Head over to [releases](https://github.com/ethanhung11/ChoanoCounter/releases) and download the installer for your platform.

### Windows
Simply run the exe file!

### macOS
Extract and move ChoanoCounter.app to Applications.

If macOS warns that the app is from an unidentified developer:
1. Cmd + Right-click the app
2. Select Open
3. Click Open again

This may not work on newer MacOS versions. If so,
1. go to System Settings > Privacy & Security, and scroll to the Security section.
2. Click "Open Anyway" next to "'ChoanoCounter' was blocked to protect your Mac."
See [this YouTube video](https://www.youtube.com/watch?v=biIvAM94b98) for more detail.

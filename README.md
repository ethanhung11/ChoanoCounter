# ChoanoCounter

For all your choano counting needs!

## Installation

### Windows
Download and run:
ChoanoCounter-vX.Y.Z-Windows-Setup.exe

### macOS
Download:
ChoanoCounter-vX.Y.Z-macOS.zip

Extract and move ChoanoCounter.app to Applications.

If macOS warns that the app is from an unidentified developer:

1. Right-click the app
2. Select Open
3. Click Open again
